Supported Linux Distribution

If you do not need recent features, we recommend you use the packages provided by your Linux distribution. RobOptim is available on the following Linux Distributions:

  • Arch Linux
  • Fedora Core
  • Debian
  • Ubuntu

RobOptim is also available as a RobotPkg package.
